# atlas-funec
App demonstrativo criado durante palestra na FUNEC.

### Páginas

- Lista de Países
- Informações do País

### Plataformas

- Mobile (Cordova + Ionic View App)
- Web (Surge.sh)
- Desktop (Electron)

### Build

Android: `ionic build android`
Web (surge): `surge --domain atlas-funec.sh`
Desktop: `npm run electron`
